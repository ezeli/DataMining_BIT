# DataMining_BIT
> BIT2020 数据挖掘课程作业

## 互评作业3：分类与预测
### 选用数据集
[Video Game Sales](https://www.kaggle.com/gregorut/videogamesales)  
包含11个属性，16598行数据。

### 所探索的问题
 1. 电子游戏市场分析：受欢迎的游戏、类型、发布平台、发行人。
 2. 预测每年电子游戏销售额。
 3. 可视化应用。
